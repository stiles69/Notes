<u>Volumes</u> /dev/sdc? /path/to/mnt/point btrfs defaults01</b> Do not setup compress on the whole
	device, instead use compress=lzo for subvolumes you want to compress.
<u>Subvolumes</u> /dev/sdc? /Path/To/Mnt/Pointbtrfs defaults,compress=lzo,subvolid=?01</b>
To get the subvolid you need to manually mount main btrfs partition first then get the subvoid's with the following
command: $ btrfs subvolume list -p /path/to/mount/</b>
It will return something like: ID 257 gen 378 parent 5 top level 5 path Media ID 258 gen
	356 parent 5 top level 5 path Data
