#How to install Arch Linux on RaspberryPi
Start fdisk to partition the SD card:
<div style='padding-left: 30pt'>
    fdisk /dev/sdX

    At the fdisk prompt, delete old partitions and create a new one:
    <div style='padding-left: 30pt'>
        Type o. This will clear out any partitions on the drive.
        Type p to list partitions. There should be no partitions left.
        Type n, then p for primary, 1 for the first partition on the drive, press ENTER to

        <div style='padding-left: 60pt'>
            accept the default first sector, then type +100M for the last sector.

            <div style='padding-left: 30pt'>
                Type t, then c to set the first partition to type W95 FAT32 (LBA).
                Type n, then p for primary, 2 for the second partition on the drive, and then press

                <div style='padding-left: 60pt'>
                    ENTER twice to accept the default first and last sector.

                    <div style='padding-left: 30pt'>
                        Write the partition table and exit by typing w.

                        Create and mount the FAT filesystem:
                        <div style='padding-left: 30pt'>
                            mkfs.vfat /dev/sdX1
                            mkdir boot
                            mount /dev/sdX1 boot

                            Create and mount the ext4 filesystem:
                            <div style='padding-left: 30pt'>
                                mkfs.ext4 /dev/sdX2
                                mkdir root
                                mount /dev/sdX2 root

                                Download and extract the root filesystem (as root, not via sudo):
                                <div style='padding-left: 30pt'>
                                    wget <a href="http://os.archlinuxarm.org/os/ArchLinuxARM-rpi-latest.tar.gz" title="http://os.archlinuxarm.org/os/ArchLinuxARM-rpi-latest.tar.gz"
                                        class="http">http://os.archlinuxarm.org/os/ArchLinuxARM-rpi-latest.tar.gz (this
                                        is the ARMv6 verison
                                        for RPi1 and 0)
                                        OR
                                        wget <a href="http://os.archlinuxarm.org/os/ArchLinuxARM-rpi-2-latest.tar.gz"
                                            title="http://os.archlinuxarm.org/os/ArchLinuxARM-rpi-2-latest.tar.gz"
                                            class="http">http://os.archlinuxarm.org/os/ArchLinuxARM-rpi-2-latest.tar.gz
                                            (this is ARMv7 version for
                                            RPi2/3)
                                            OR
                                            wget <a href="http://os.archlinuxarm.org/os/ArchLinuxARM-rpi-3-latest.tar.gz"
                                                title="http://os.archlinuxarm.org/os/ArchLinuxARM-rpi-3-latest.tar.gz"
                                                class="http">http://os.archlinuxarm.org/os/ArchLinuxARM-rpi-3-latest.tar.gz
                                                (ARMv8 x64 version for v3
                                                ONLY, not fully supported)
                                                bsdtar -xpf ArchLinuxARM-&lt;rpi-version&gt;-latest.tar.gz -C root
                                                sync

                                                Move boot files to the first partition:
                                                <div style='padding-left: 30pt'>
                                                    mv root/boot/* boot

                                                    Unmount the two partitions:
                                                    <div style='padding-left: 30pt'>
                                                        umount boot root

                                                        Insert the SD card into the Raspberry Pi, connect ethernet, and
                                                        apply 5V power.
                                                        Use the serial console or SSH to the IP address given to the
                                                        board by your router. The default root password is
                                                        'root'.
