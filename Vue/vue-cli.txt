vue init egoist/vuepack &lt;projectname&gt;</b>
Makes a new project based on github &lt;egoist/vuepack&gt; template.



vue init &lt;template&gt; &lt;projectname&gt;</b>
Makes a new vue project
Default Official templates are:
<ul>
	browerify</li>
	browserify-simple</li>
	pwa</li>
	simple</li>
	webpack</li>
	webpack-simple</li>
</ul>



vue list</b>
