Create and manage volumes


Created Monday 07 May 2018



Unlike a bind mount, you can create and manage volumes outside the scope of any container.



Create a volume:



$ docker volume create my-vol</b>
List volumes:



$ docker volume ls</b>



local my-vol
Inspect a volume:



$ docker volume inspect my-vol</b>

            Remove a volume:



            $ docker volume rm my-vol</b>
