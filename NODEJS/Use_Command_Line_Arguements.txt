====== Use Command Line Arguements ======
Created Tue 05 February 2019

Vanilla javascript argument parsing:
const args = process.argv;
console.log(args);
This returns:

$ node server.js one two=three four
['node', '/home/server.js', 'one', 'two=three', 'four']
