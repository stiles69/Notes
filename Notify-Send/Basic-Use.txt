<div class="zim-object">
  <pre><code class="sh">notify-send Test "Hello World"</code></pre>


  The --help option shows basic usage:
  $notify-send --help</b>
  Usage:
  notify-send [OPTION...] &lt;SUMMARY&gt; [BODY] - create a notification



  Help Options:
  -?, --help Show help options



  Application Options:
  -u, --urgency=LEVEL Specifies the urgency level (low, normal, critical).
  -t, --expire-time=TIME Specifies the timeout in milliseconds at which to expire the notification.
  -i, --icon=ICON[,ICON...] Specifies an icon filename or stock icon to display.
  -c, --category=TYPE[,TYPE...] Specifies the notification category.
  -h, --hint=TYPE:NAME:VALUE Specifies basic extra data to pass. Valid types are int, double, string and byte.
  -v, --version Version of the package.





  <div style='padding-left: 30pt'>
    Some examples:




    <ol style='padding-left: 30pt' type="1" start="1">
      Basic message with an icon in front of the Title showing 5000 milliseconds</li>
    </ol>
    <div style='padding-left: 30pt'>
      Code:




      <div style='padding-left: 30pt'>
        notify-send "Message Title" "The message body is shown here" -i /usr/share/pixmaps/idle.xpm -t 5000




        <ol style='padding-left: 30pt' type="1" start="2">
          Show contents of a file:</li>
        </ol>
        <div style='padding-left: 30pt'>
          Code:




          <div style='padding-left: 30pt'>
            notify-send test "`tail /var/log/syslog`"




            <ol style='padding-left: 30pt' type="1" start="3">
              Follow log file:</li>
            </ol>
            <div style='padding-left: 30pt'>
