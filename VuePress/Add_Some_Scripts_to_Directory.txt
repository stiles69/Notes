Add Some Scripts to Directory


Created Tuesday 15 May 2018


<div class="zim-object">
  <pre><code class="sh">{
  "scripts": {
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs"
  }
}</code></pre>


  You can now start writing with:
  yarn docs:dev # OR npm run docs:dev</b>



  To generate static assets, run:
  yarn docs:build # Or npm run docs:build</b>
